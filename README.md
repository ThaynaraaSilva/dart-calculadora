# 🧮 Dart Calculadora (Terminal)

Uma calculadora simples desenvolvida em **Dart**, executada diretamente pelo terminal.
O projeto foi criado com o objetivo de praticar lógica de programação, entrada de dados e execução de operações matemáticas usando Dart.

---

## 📌 Sobre o projeto

Esta aplicação permite que o usuário digite operações matemáticas livremente no terminal e receba o resultado imediatamente.

A calculadora interpreta expressões digitadas pelo usuário, possibilitando cálculos rápidos e interativos.

---

## ⚙️ Funcionalidades

* ➕ Soma
* ➖ Subtração
* ✖️ Multiplicação
* ➗ Divisão
* 🧠 Suporte a expressões matemáticas (ex: `2 + 2 * 3`)
* 🚪 Comando para encerrar o programa

---

## 💻 Como executar

### 1. Clonar o repositório

```bash
git clone https://github.com/ThaynaraaSilva/dart-calculadora.git
```

### 2. Entrar na pasta do projeto

```bash
cd dart-calculadora
```

### 3. Instalar dependências

```bash
dart pub get
```

### 4. Executar a calculadora

```bash
dart run bin/calculator.dart
```

---

## 🧪 Exemplo de uso

```
Digite uma operação: 8 + 2
Resultado: 10

Digite uma operação: 10 / 5
Resultado: 2

Digite uma operação: exit
Aplicação finalizada.
```
